1. Πως μπορούμε να κάνουμε ένα Request σε ένα εξωτερικό API χωρίς να γνωρίζει ολόκληρο το url μας;
   

   -Δεν γίνεται να κάνω κάποιο request σε ένα εξωτερικό API χωρίς να γνωρίζω το full URL. Το URL είναι το  address της πηγής που θέλουμε να φτάσουμε  
    και είναι ανγκαίο κομμάτι για να πετύχουμε ένα request.Αν δε το έχουμε δεν μπορούμε να έχουμε πρόσβαση στα recourses του.



2. Αν κάνω ενα GET request και ζητήσω ενα jpeg image, το response τι  content-type πρεπει να εχει;
  
    -Το content-type ενός GET request σε ένα jpeg image θα πρέπει να γυρίσει σα response image/jpeg.


3. Πως μπορώ να κάνω ένα GET request και το περιεχόμενο του να κρατηθεί στην cache για 1 μέρα;

    -Για να κάνω ένα αίτημα GET request που διαρκεί 1 ημέρα, μπορούμε να να βάλουμε ενα Cache-Control που να έχει κάποιο age.



4. Μπορώ να έχω μια εφαρμογή που κάνει μόνο GET requests;

    -Μπορώ να έχω μια εφαρμογή που κάνει μόνο Get requests καθώς την GET request την χρησιμοποιώ για να λαμβάνω πληροφορίες και υλικό από έναν server 
     όπως μία ιστοσελίδα , να δω μια φωτογραφία ή κάποιο file. Ένα τέτοιο πρόγραμμα θα μπορούσε να χρησιμποιηθεί για παράδειγμα στο να τραβάει data από 
     ένα API, να κάνω display πληροφορίες και διάφορα άλλα.


5. Μπορώ να έχω μια εφαρμογή που κάνει μόνο POST requests;


    -Αντίστοιχα και την προηγουμένη ερώτηση μπορώ να έχω μια εφαρμογή που να κάνει μόνο POST requests. Για παράδειγμα  μπορεί να χρησιμποιηθεί κάτι τέτοιο 
     στο να δημιουργηθεί μία φόρμα που ζητάει να συμπληρώσεις το βιογραφικό σου. Επίσης μέσω των POST requests μπορούμε να ανεβάζουμε πλροφορίες σε έναν server.


6. Τι status code περιμένω από ένα API αν δεν έχω στείλει ένα υποχρεωτικό πεδίο;


    -Το status code που περιμένω απο ένα API που δεν του έχω στείλει ένα υποχρεωτικό πεδίο είναι 400 Bad REquest. Το συγκεκριμένο status code αναφέρει πως έγινε ένα 
     request αλλά ο server δεν μπορούσε να το κάνει process εφόσον λείπει μία υποχρεωτική πληροφορία.


7. Τι status code περιμένω από ένα API αν δεν υπάρχει επικοινωνία με τη βάση; 

    -Το status code που περιμένω απο ένα API που δεν έχει επικοινωνία με τη βάσει είναι το 500 Internal Error. Το status αυτό υποδηλώνει οτι υπήρξε ένα unexpected condition
     όπου στο παράδειγμα μας είναι η αποτυχία επικοινωνίας με τη βάση. Το API μπορεί να γυρίσει επίσης και μια πλροφορία για το τι πήγε συγκεκριμένα λάθος.


8. Μπορώ να κάνω Login με GET request;

    -Δεν είναι συνετό να χρησιμποιούμε Get requests για login. Τα Get requests τα χρησιμοποιούμαι συνήθως για να κάνουμε retrieve-display informations. Αντίθετα για να κάνουμε ένα login
     πρέπει να χρησιμοποιήσουμε ένα Post request.

9. Ποια είναι η διαφορά ανάμεσα στο PUT και στο POST;

   -Η διαφορά ανάμεσα σε ένα PUT και ένα POST είναι το PUT χρησιμοποιείται για να κάνουμε update ένα ήδη υπάρχον αντικείμενο στη βάση ενώ το POST χρησιμοποιείται για να δημιουργήσουμε ένα
    νέο object.


10. Ποια μέθοδο θα πρέπει να χρησιμοποιήσω για να «ανεβάσω» ένα αρχείο;

    -Για να ανεβάσουμε ένα αρχείο θα χρησιμοποιούμε ένα POST request όπου μέσα σαν Request Body θα έχει το αρχείο με ta δεδομένα που θέλω να κάνω upload.
